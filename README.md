## 📌 Project Overview

A mobile application that simplifies event discovery and ticket booking. Users can browse events by categories, book tickets, and track bookings.

## 🛠️ Tools Used
- **Figma** – UI design, wireframing, prototyping
- **Iconify** – Icon library
- **Auto Layout** – Responsive design
- **Smart Animate** – Smooth transitions
